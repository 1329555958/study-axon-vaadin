Sample Axon projects
=====

Tools: Eclipse with M2Eclipse, Egit, WikiText, ...

Building
------

Check out this repository, e.g. under Eclipse with:

        File > Import... > Git > Projects from Git > Clone URI
using the URL:

        https://github.com/atao60/study-axon-vaadin
